Input

Biaya Stockbit

Fee jual total dipakai sebagai potongan jual. Pajak hanya tampil sebagai rincian agar tidak dihitung dua kali.

Parameter TP dan SL Tambahan

Mode lama tetap aktif jika TP memakai nominal Rupiah lama dan SL memakai R:R. Mode persen memakai alokasi per trade sebagai basis.

Auto Alokasi per Trade

Gunakan auto alokasi jika ingin lot menyesuaikan entry dan posisi TP/SL. Untuk mode target profit, TP harus berupa tick atau harga. Untuk mode risiko SL, SL harus berupa tick atau harga.

Cek Area ARA/ARB Harian

Isi close kemarin untuk melihat apakah entry, TP, dan SL masih berada dalam batas ARA/ARB hari tersebut.
